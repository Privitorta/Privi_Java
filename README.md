## Privi_C

Esercizi di programmazione in Java tratti da tracce d'esame di Programmazione (Laboratorio), A.A. 2024/25, prof. Matteo Luperto (UNIMI, SSRI)

### Compilazione

1. **Apri il terminale** nella cartella giusta
2. **Compila** i file .java con:
   ```
   javac *.java
   ```
   oppure in maniera specifica `javac nomefile.java` sostituendo `nomefile` con il nome del file che vuoi compilare
4. **Esegui** il programma:
     ```
   java Solver
     ```
   sostituendo `Solver` con il nome della classe che contiene il metodo main
